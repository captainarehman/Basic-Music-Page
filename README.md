## What's in the file?
  * Well-commented html file
  * Everything hosted online/On the Web, i.e you can just take the file and upload it/use
  * Used Inline CSS rather than internal for better navigational changes (**Note:** Some parts of the code has internal css tags so do make sure to run through the code once if you make changes
